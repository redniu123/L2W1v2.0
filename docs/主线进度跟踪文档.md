## 2. 主线目标回顾（来自实验设计书）

### 2.1 论文主线
- `GCR`：传统全局置信度弱基线
- `BAUR`：核心轻量预算感知路由器
- `DAR`：领域增强路由（对照 / 消融）
- `SH-DA++`：BAUR + 领域软提示词 + 严格回填 + 熔断

### 2.2 实验主线
- **M1-M3**：比路由器（GCR / BAUR / DAR）
- **M4-M5**：比系统（BAUR-only / SH-DA++）
- **V1-V4**：在固定最优 Router / 系统设置下做跨模型验证

### 2.3 结果主线
所有正式实验都必须：
- 保存逐样本 JSONL；
- 保存 summary；
- 保存 config snapshot；
- 可追溯到 run_id / Prompt 版本 / Router 版本。

**状态**：已完成（第一轮）

- `BAUR-only`：只保留 BAUR 路由主链

- `BAUR`

- `SH-DA++`

> “完整 SH-DA++ 是否优于仅有 BAUR 的基础链路？”

- `GCR`
- `BAUR`
- `DAR`
- `BAUR-only`
- `SH-DA++`

其中：
- `BAUR` / `DAR` 更偏 Router 对比
- `BAUR-only` / `SH-DA++` 更偏系统对比
