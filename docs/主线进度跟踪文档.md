## 2. 主线目标回顾（来自实验设计书）

### 2.1 论文主线
- `GCR`：传统全局置信度弱基线
- `BAUR`：核心轻量预算感知路由器
- `DAR`：领域增强路由（对照 / 消融）

- `DAR`
- `BAUR-only`
- `SH-DA++`

其中：
- `BAUR` / `DAR` 更偏 Router 对比
- `BAUR-only` / `SH-DA++` 更偏系统对比
